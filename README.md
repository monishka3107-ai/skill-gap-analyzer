📖 Overview

Choosing a career path often raises an important question:

“What skills do I need to reach my dream role?”

Skill Gap Analyzer solves this problem by comparing a user’s existing skills with the requirements of a target role and providing actionable recommendations for improvement.

The application analyzes the gap, identifies missing skills, and generates a structured roadmap to help users upskill efficiently.

⸻

✨ Features

✅ Analyze current skills against target career roles

✅ Identify missing or weak skills

✅ Generate personalized learning roadmaps

✅ Suggest prerequisite topics for learning

✅ Recommend learning resources

✅ User-friendly Streamlit interface

✅ Instant skill gap assessment

⚙️ Installation

Clone the repository:

git clone https://github.com/monishka3107-ai/skill-gap-analyzer.git

Navigate to the project folder:

cd skill-gap-analyzer

Install dependencies:

pip install -r requirements.txt

Run the application:

streamlit run app.py
